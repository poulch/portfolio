O mnie

Witam, mam na imię Paweł i jestem front-end developerem. Przygodę z programowanie rozpocząłem na początku 2016 roku stażem w firmie Inwave. Podczas stażu miałem możliwość pracy z Html 5, Css 3 oraz technologiami backendowymi Php, Zend Framework. Po stażu zostałem zatrudniony jako frontend developer i zacząłem naukę javascript na początku ES5 a trochę później ES6. Na co dzień zajmuję się programowanie warstwy wizualnej stron internetowych. Wykorzystuje takie narzędzia jak: Html 5, Css 3, Sass, Javascript, Jquery, Gulp, biblioteki javascriptowe takie jak: Magnific Popup, Flickity, Packery itp. Od wakacji uczę się Reacta wraz z całym ekosystemem (React Router, Redux, Redux-thunk). Wiedzę z Reacta wykorzystuje przy budowie aplikacji Quiz Designer - którą buduję wraz z kilkoma znajomymi programistami. Quiz Designer to aplikacja open source służąca do projektowania i przeprowadzania quizów. Aplikacja wykorzystuje na froncie React + Redux a za warstwę backendową Node / Express + MongoDb.
